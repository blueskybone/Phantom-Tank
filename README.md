# Phantom-Tank
幻影坦克，命令行

使用方法：控制台进入exe所在文件夹，将待合成的两张图片放入同一目录。命令格式为
```
tank file1 file2 result
```
file1、file2 为必选参数，file1为表图，file2为里图

result 为可选参数，缺省值为result.png
